# Fontes gratuitas do ADQL Analytics Layer

## Prioridade operacional

1. **FBref via soccerdata**: estatísticas atuais de equipes e jogadores.
2. **Understat via soccerdata**: xG, xA e finalizações em competições cobertas.
3. **Football-Data.co.uk**: resultados, odds e estatísticas básicas em CSV.
4. **football-data.org**: fixtures, standings e scorers via API.
5. **StatsBomb Open Data**: estudo avançado com eventos abertos.
6. **ClubElo via soccerdata**: rating de força relativa.
7. **TheSportsDB**: metadados, eventos, equipes e imagens.

## Regras de uso

- Nunca trate número isolado como conclusão.
- Cruzar estatística com vídeo sempre que possível.
- Registrar fonte e data de consulta nos relatórios.
- Usar cache local.
- Evitar scraping agressivo.
- Separar dado factual de interpretação.

## Integração com ADQL UI

- C-03: cenas táticas, mapas e zonas.
- C-04: radar coletivo.
- C-05: comparação entre jogadores.
- C-06: tabelas.
- C-07: capas editoriais com contexto.
